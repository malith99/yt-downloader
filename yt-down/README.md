
- **Installation**

```shell
$ git clone https://github.com/malith99/yt-downloader.git
```

- **Change directory**

```shell
$ cd yt-downloader
```

- **Virtual Environment** `Optional`

```shell
$ pip install virtualenv
$ virtualenv venv
$ source venv/bin/activate
```

- **Install dependencies**

```shell
$ pip install -r requirements.txt
```

- **Run the Bot**

```shell
$ python main.py
```
